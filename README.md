# PitchSync (MERN)

PitchSync is a dual-role cricket turf booking and match management platform.

## Tech

- **Client**: Vite + React + React Router + Axios
- **Server**: Node.js + Express + MongoDB (Mongoose) + JWT + bcrypt

## Setup

### 1) Server

```bash
cd server
copy .env.example .env
npm install
npm run dev
```

### 2) Client

```bash
cd client
copy .env.example .env
npm install
npm run dev
```

## Default URLs

- **API**: `http://localhost:5000/api/health`
- **Web**: `http://localhost:5173`
